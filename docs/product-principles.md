# Product Principles

- EmpirePicks is a decision-support tool for line shopping and fair price evaluation.
- Public UI must answer: what is fair, what is best, what is actionable, and why it ranks where it does.
- Tables outrank cards. Dense clarity outranks decorative space.
- Internal diagnostics stay separate from public product language.
- Copy stays short, operational, and betting-native.
- Real history is shown only when persisted data exists.
